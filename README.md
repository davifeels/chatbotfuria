## 🚀 Visão Geral do Projeto

A **Basilío Joias Exclusivas** é uma vitrine digital de e-commerce que demonstra a criação de uma **interface de usuário (UI) elegante e responsiva** para uma loja de joias de luxo. O projeto foca em proporcionar uma **experiência visual premium** para o usuário, destacando os produtos de forma atraente e intuitiva.

Este trabalho é um exemplo prático de como construir um frontend limpo, moderno e funcional utilizando apenas as tecnologias essenciais da web: HTML, CSS e JavaScript. A estrutura modular do código e a atenção aos detalhes visuais são pontos chave deste projeto.

## ✨ Funcionalidades Principais

* **Design de E-commerce Elegante:** Interface visual projetada para o segmento de luxo, com paleta de cores sofisticada (preto, dourado, branco) e tipografia refinada (`Montserrat`, `Lora`, `Roboto`).
* **Catálogo de Produtos:** Exibição organizada de joias por categorias (Anéis, Brincos, Colares, Pulseiras), com cards de produtos claros e informativos (nome, descrição, preço).
* **Efeitos de Interação (Hover):** Cards de produtos com efeitos de elevação e aumento sutil da imagem ao passar o mouse, proporcionando uma experiência de usuário dinâmica.
* **Navegação por Categoria:** Páginas dedicadas para cada tipo de joia, permitindo que os usuários explorem produtos específicos facilmente.
* **Estrutura HTML Semântica:** Uso de tags HTML5 apropriadas para garantir acessibilidade e SEO (Search Engine Optimization).
* **Estilização Profissional com CSS3:** Aplicação de conceitos como Flexbox para layouts flexíveis, transições suaves e sombras sutis para uma estética moderna.
* **Interatividade Básica com JavaScript:** Gerenciamento de eventos de clique para navegação entre páginas de categorias.
* **Rodapé Completo:** Informações de direitos autorais e links para políticas (Ex: Política de Privacidade, Termos de Uso).

## ⚙️ Como Executar Localmente

Siga os passos abaixo para clonar o repositório e rodar o projeto em seu ambiente de desenvolvimento local:

1.  **Clone o Repositório:**
